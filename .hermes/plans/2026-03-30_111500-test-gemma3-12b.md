# Testplan: Gemma3 12B på RTX 3080

**Dato:** 2026-03-30  
**Maskinvare:** RTX 3080 (10GB VRAM)  
**Modell:** google/gemma-3-12b-it (Q4_K_M eller tilsvarende kvantisert)  
**Baseline:** Gemma3 4B (29.4% → 40.9% etter våre forbedringer)

---

## Mål

1. **Mål norsk-score ≥ 60%** (opp fra 40.9% med 4B)
2. **Teste VRAM-bruk** - sikre at 12B kjører stabilt på RTX 3080
3. **Validere system prompt-effekt** - større modell = bedre instruksjons-følging
4. **Optimalisere temp/max_tokens** for 12B-modellen
5. **Dokumentere responstid** - er den fortsatt rask nok for chat?

---

## Forberedelser

### Sjekkliste før testing:
- [ ] Modell ferdig lastet ned
- [ ] LM Studio konfigurert med 12B-modell
- [ ] Server/startet på port 3000
- [ ] minnebruk ved oppstart notert
- [ ] Kontekstlengde satt til 4096 eller 8192

### Forventet VRAM-bruk (estimat):
- Gemma3 12B Q4_K_M: ~7-8GB VRAM
- RTX 3080 10GB: Skal ha nok med margin
- Hvis OOM: Prøv Q3_K_L eller lavere kontekst

---

## Fase 1: Baseline Test (15 min)

**Mål:** Etablere norsk-score med 12B uten system prompt

```bash
# Test med default LM Studio settings (ingen system prompt)
python3 test_12b_baseline.py
```

**Test prompts (samme som før):**
- 20 representative prompts fra alle kategorier
- Måle norsk-score
- Måle responstid (ms)
- Sammenligne med 4B-baseline (40.9%)

**Forventet resultat:**
- Uten system prompt: ~35-45% (større modell, men ingen norsk instruksjon)
- Responstid: ~2-4 sekunder (lengre enn 4B)

---

## Fase 2: System Prompt Test (20 min)

**Mål:** Teste om 12B følger vår norske system prompt bedre enn 4B

```bash
# Med vår system_prompt.txt lastet
python3 test_12b_with_prompt.py
```

**Hva testes:**
1. Bruker den norske ordlisten? (kjempe, supert, flott, skjønner)
2. Gjenkjenner den dialekt-uttrykk? (kjekt, tøft, rått)
3. Er tonen naturlig og uformell?
4. Unngår den engelske ord?

**Forventet resultat:**
- Med system prompt: **55-70%** norsk-score
- Betydelig bedre enn 4B (40.9%)
- Mer naturlige, varierte responser

---

## Fase 3: Temperature & Max_tokens Optimalisering (25 min)

**Mål:** Finne beste innstillinger for 12B-modellen

### Test matrise:
| Temperature | Max Tokens | Brukssituasjon |
|-------------|------------|----------------|
| 0.5 | 150 | Korte hilsener |
| 0.7 | 200 | Standard chat |
| 0.8 | 250 | Lengre forklaringer |
| 0.9 | 300 | Kreative svar |

**For hver kombinasjon:**
- 10 test prompts
- Måle norsk-score
- Måle responstid
- Vurdere responskvalitet (1-5)

**Forventet resultat:**
- 12B tåler høyere temperatur uten å bli "gal"
- Optimal: temp=0.7-0.8, max_tokens=200-250

---

## Fase 4: Dialekt & Kultur Test (20 min)

**Mål:** Teste spesifikke norske uttrykk og kulturforståelse

### Testcases:
```python
dialect_tests = [
    ("Dette var kjekt!", "Skal gjenkjenne 'kjekt'"),
    ("Skikkelig tøft!", "Skal gjenkjenne 'tøft'"),
    ("Helt rått!", "Skal gjenkjenne 'rått'"),
    ("Morn!", "Skal svare på 'morn'"),
    ("Kvelden!", "Skal svare på 'kvelden'"),
]

culture_tests = [
    ("Hva er best med Norge?", "Naturen, folket, osv."),
    ("Er du glad i brunost?", "Skal vite hva brunost er"),
    ("17. mai?", "Grunnlovsdagen, feiring"),
    ("Påskeferie", "Skal forstå norsk påske"),
    ("Koselig", "Sentralt norsk konsept"),
]
```

**Forventet resultat:**
- Dialekt: **80-100%** forståelse (med vår handler: 100%)
- Kultur: **60-80%** gode svar
- 12B skal ha mer kunnskap om Norge i treningssettet

---

## Fase 5: Kalender & Praktisk Test (15 min)

**Mål:** Teste om 12B forstår kalenderkommandoer bedre

### Testcases:
- "Husk å kjøpe melk i morgen"
- "Påminn meg om å ringe mamma i kveld"
- "Jeg har møte på torsdag kl 14"
- "Hva skjer neste uke?"

**Hva sjekkes:**
- Forstår den at dette er kalender-oppgaver?
- Gir den naturlige bekreftelser? ("Notert!", "Fikk det!")
- Slår den ikke over til generell AI-respons?

---

## Fase 6: Ytelse & Stabilitet (15 min)

**Mål:** Sikre at 12B kjører stabilt på RTX 3080

### Tester:
1. **Sammenhengende chat:** 20 meldinger på rad
2. **Lang kontekst:** 2000+ tokens kontekst
3. **Hukommelse:** Ingen minnelekkasje over tid
4. **Responstid:** Maks 5 sekunder per svar

**Monitorering:**
- `nvidia-smi` under testing
- LM Studio RAM/VRAM visning
- Tid per respons

## Neste steg etter testing

**Hvis suksess (≥50% norsk):**
1. Oppdater default config til å bruke 12B-innstillinger
2. Utvid system prompt med flere norske nyanser
3. Legg til flere dialekt-uttrykk i handler

**Hvis problemer (OOM, treg):**
1. Prøv Q3_K_L kvantisering
2. Reduser kontekstlengde til 4096
3. Vurder 8B-modell som kompromiss

**Hvis fantastisk (≥65% norsk):**
1. Vurder å legge til nynorsk-støtte
2. Eksperimenter med enda lengre system prompts
3. Lag "personlighets-profiler" (formell/uformell)
